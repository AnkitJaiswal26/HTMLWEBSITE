Introduction To HTML

  1. HTML stands for HyperText Markup Language
  2. HTML is the satndard markup language for Webpages.
  3. HyperText is ordinary Text with extra features such as formatting, images, multimedia and links to other documents
  4. Markup is the process of taking ordinary text and adding extra symbols such as editor's proof, reading symbols. Each of the sumbols tells the browser how to display the text.
  5. HTML is essentially a set of instructions to web browsers for formatting and layout of webpage


    6. HTML was originally developed by Tim Berners-Lee in 1990. He is also known as the father of the web. In 1996, the World Wide Web Consortium (W3C) became the authority to maintain the HTML specifications. HTML also became an international standard (ISO) in 2000. HTML5 is the latest version of HTML. HTML5 provides a faster and more robust approach to web development.

Features of HTML

  • HTML is a set of codes that is used to create document and then it can be published on the World Wide Web.
  • HTML supports to frames including target window and borderless frames.
  • It contains powerful formatting facilities for text, page, images, etc.
  • It supports image format such as .png ( Portable Network Graphics ), .bmp( Bitmap ), .jpeg( Joint Photographic Experts Group ) and animated .gif( Graphics Interchange Format )
  • HTML support forms which makes it possible to create documents that collects and processes user's input.
