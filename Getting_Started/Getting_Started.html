What will you learn in this tutorial ?

    In this tutorial you will learn how easy it is to create an HTML document or a web page. To begin coding HTML you need only two stuff: a simple-text editor and a web browser. Well, let's get started with creating your first HTML page.

Text Editor for HTML

    A text editor is a type of computer program that edits plain text.
    In most Operating Systems text editor is pre-installed.
    Following are the differnt text editors in different operating systems.
  1. Windows OS: Notepad or Wordpad.
  2. Macintosh OS: Simple Text.
  3. Unix OS: Pico.

Web Browsers

    A web browser is a software application for accessing information on the World Wide Web.
    Following are the names of mostly used Web Browsers.
  • Google Chrome
  • Mozilla Firefox
  • Internet Explorer
