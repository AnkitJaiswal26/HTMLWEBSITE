HTML Tag

  • A Tag is a single unit of Markup Language.
  • Tags are instructions that are written directly into text edition.
  • Tags are started with '<' (less than sign) followed by a keword and ends with '>' (greater than sign).
  • There are two types of tags in HTML
    1. Paired Tags
    2. Unpaired or Stand-alone Tags
    We will learn about paired and unpaired tags in detail later.

    For now just remember that paired tags are the one that have a opening tag and a closing tag.
